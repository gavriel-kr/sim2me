# Sim2Me eSIM

eSIM store for travel: browse destinations, compare plans, and buy data packages. Built with Next.js 14, TypeScript, Tailwind CSS, and next-intl (EN / HE / AR).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

See **[DEPLOYMENT.md](./DEPLOYMENT.md)** for:

1. Pushing this project to GitHub  
2. Connecting the repo to Vercel and deploying  
3. Optional custom domain setup  

## Stack

- **Framework:** Next.js 14 (App Router)  
- **i18n:** next-intl (locale-based routing)  
- **Styling:** Tailwind CSS  
- **State:** Zustand (cart), TanStack Query (data)
