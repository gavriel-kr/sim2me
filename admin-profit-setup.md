# רווח לפי מוצר – מה לעשות

## כבר עשיתי בשבילך (במחשב שלך)
- בסיס הנתונים מעודכן (טבלאות אגרות + עמודת עלות סים).
- הוגדרו ברירת מחדל: Paddle 5% + $0.50.

## איך לראות את זה
1. הרץ: **`npm run dev`**
2. פתח בדפדפן: **http://localhost:3000/admin**
3. היכנס (אם צריך).
4. בתפריט: **eSIM Packages** – תראה רווח נקי, מרווח ו-break-even לכל חבילה.  
   **Fees / Charges** – עריכת אגרות (Paddle + אגרות נוספות).

---

## אם יש לך שרת (פרודקשן) עם DB נפרד
**הרץ שם פעם אחת** (בתיקיית הפרויקט,משתנה DATABASE_URL מצביע ל-DB של הפרודקשן):

```bash
npm run db:ready
```

זה מעדכן את הסכמה ומוסיף את הגדרות האגרות. אחרי זה לא צריך להריץ שוב.
